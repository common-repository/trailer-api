Based on the post name this plugin will add a suitable trailer.
This plugin is a must have for every movie blog.


== Description ==


Adds to every post a movie trailer. The title of the post must be the name of the movie. The trailer will be added to the bottom of the post.
The trailer is only added if the post is displayed as a single post.

Example: The title of the post is "The Open Road".

The code that will be insered is based on the embed iframe function of youtube.


This plugin is based on the API of www.trailerapi.com.


== Installation ==


1. Upload the folder powerfull-social-bookmarking to the /wp-content/plugins/ directory

1. Activate the plugin Powerfull Social Bookmarking through the 'Plugins' menu in WordPress
